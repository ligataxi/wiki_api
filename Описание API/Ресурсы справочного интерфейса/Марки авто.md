### Получение списка марок автомобилей

### Запрос:

```java
GET /api/v1/car_vendors/
```

#### Параметры запроса:

|     |     |     |
| --- | --- | --- |
| **Название** | **Тип** | **Описание** |
| limit | int | Лимит записей в ответе |
| offset | int | Пропустить Х записей |

### Ответ:

```java
{
  "meta": <RestMetaObject>,
  "objects": [<CarVendorObject>]
}
```
