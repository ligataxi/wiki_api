#### Пример структуры

```java
{
  "operation_type": 1,
  "id": 1,
  "name": "10 грн за каждый заказ",
  "resource_uri": "/api/v1/driver_groups/1/"
}
```

#### Описание структуры

|     |     |     |
| --- | --- | --- |
| **Название поля** | **Тип данных** | **Описание** |
| operation_type | int | Тип операции |
| id  | int | ID записи |
| name | str | Название |
| resource_uri | str | Ссылка на детальный просмотр записи |
