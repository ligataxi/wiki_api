#### Пример структуры

```java
{
  "id": 1,
  "name": "Синий",
  "resource_uri": "/api/v1/car_colors/1/"
}
```

#### Описание структуры

|     |     |     |
| --- | --- | --- |
| **Название поля** | **Тип данных** | **Описание** |
| id  | int | ID записи |
| name | str | Название |
| resource_uri | str | Ссылка на детальный просмотр записи |
