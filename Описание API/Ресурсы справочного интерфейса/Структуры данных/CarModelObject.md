#### Пример структуры

```java
{
  "id": 6,
  "name": "Легковой",
  "resource_uri": "/api/v1/car_models/6/"
}
```

#### Описание структуры

|     |     |     |
| --- | --- | --- |
| **Название поля** | **Тип данных** | **Описание** |
| id  | int | ID записи |
| name | str | Название модели авто |
| resource_uri | str | Ссылка на детальный просмотр записи |
