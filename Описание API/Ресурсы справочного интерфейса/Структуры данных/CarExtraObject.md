#### Пример структуры

```java
{
  "company_id": 1,
  "id": 6,
  "name": "Верхний багажник",
  "resource_uri": "/api/v1/car_extras/6/"
}
```

#### Описание структуры

|     |     |     |
| --- | --- | --- |
| **Название поля** | **Тип данных** | **Описание** |
| id  | int | ID записи |
| name | str | Название опции авто |
| company_id | int | ID компании |
| resource_uri | str | Ссылка на детальный просмотр записи |
