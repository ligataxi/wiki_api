### Получение условий работы водителей

### Запрос:

```java
GET /api/v1/driver_rates/
```

#### Параметры запроса:

|     |     |     |
| --- | --- | --- |
| **Название** | **Тип** | **Описание** |
| limit | int | Лимит записей в ответе |
| offset | int | Пропустить Х записей |

### Ответ:

```java
{
  "meta": <RestMetaObject>,
  "objects": [<DriverRateObject>]
}
```
