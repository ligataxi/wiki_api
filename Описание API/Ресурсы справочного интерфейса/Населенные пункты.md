### Получение списка населенных пунктов

### Запрос:

```java
GET /api/v1/localities/
```

#### Параметры запроса:

|     |     |     |
| --- | --- | --- |
| **Название** | **Тип** | **Описание** |
| limit | int | Лимит записей в ответе |
| offset | int | Пропустить Х записей |

### Ответ:

```java
{
  "meta": <RestMetaObject>,
  "objects": [<LocalityObject>]
}
```

|     |     |     |
| --- | --- | --- |
| **Название** | **Тип** | **Описание** |
| locality_id | int | id записи |
