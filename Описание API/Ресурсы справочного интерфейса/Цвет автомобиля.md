### Получение списка

### Запрос:

```java
GET /api/v1/car_colors/
```

#### Параметры запроса:

|     |     |     |
| --- | --- | --- |
| **Название** | **Тип** | **Описание** |
| limit | int | Лимит записей в ответе |
| offset | int | Пропустить Х записей |

### Ответ:

```java
{
  "meta": <RestMetaObject>,
  "objects": [<CarColorObject>]
}
```

```java
GET /api/v1/car_colors/<car_color_id>/
```

|     |     |     |
| --- | --- | --- |
| **Название** | **Тип** | **Описание** |
| car_color_id | int | id записи |

```java
<CarColorObject>
```
