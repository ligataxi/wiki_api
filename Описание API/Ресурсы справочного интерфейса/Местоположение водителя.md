### Получение текущего местоположения водителя

### Запрос:

```java
GET /api/v1/drivers/<call_sign>/location/
```

#### Параметры запроса:

|     |     |     |
| --- | --- | --- |
| **Название** | **Тип** | **Описание** |
| call_sign | int | Позывной водителя |

### Ответ:

```java
{
  <LocationObject>
}
```
